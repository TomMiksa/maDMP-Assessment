An extension of the DMP Common Standard Ontology to address the information required by the Science Europe DMP Evaluation Rubric.: Description back to ToC

This ontology aims to add additional information to the DCS standard to better cover funder requirements stated by the Science Europe DMP Evaluation Rubric.
